# BlockChainSim
Simulates the generation of the next hash in a Block-Chain algorithm

1) Run "make". This will generate Newman_Jared_BTC executable.

-Newman_Jared_BTC [blockChainFile] [transactionFile]	
	Basic Mining simulation program. First arguement is the file containing previous blocks in the block chain, 
	the second is a file containing all the transactions in the block chain.

	ex: ./Newman_Jared_BTC blockchain_1.txt transactions_1.txt

2) Run "make clean" to remove the executable file.
